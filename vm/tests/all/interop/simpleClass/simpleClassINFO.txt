//simpleClass.txt

'simpleClass' is an example for polyglot language interoperability, based on a common data struct/class. 

'simpleClass.*' contains a program itself (='main method'), 'simpleClassPoint.*' contains the data structure needed. 

Point
-----------
>integer x
>integer y
-----------
>int() getX(), getY()
>void setX(int), setY(int)
>double distanceTo(Point other)
>void printMe()
